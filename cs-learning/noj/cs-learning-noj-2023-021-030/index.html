# T026 最大数字

从 n 开始逐项遍历。对数字进行拆分，拆分结束之后逐位比较，找到最大符合要求的数字即可。

# T027 竖式乘法

从来没见过这么离奇的题目，不过实际做下来还挺有意思的。

算法用到了整数位数判断，整数单位取出等技巧，比价繁琐的主要在于绘图模块，逻辑判断较多。

该算法还有优化空间，在此处不进行进一步优化，能 AC 即可。

# 关于 BFS

BFS-Breath First Search 广度优先算法，与之对应的是 DFS-Deep First Search 深度优先算法。

BFS 本质是一个流程表，将分支树中与某个节点相连的所有结点添加到流程表中，等待未来的计算。由于 BFS 本身的特性，由其产生的路径必然是起点与终点间的最短路径，但代价是空间复杂度较大。
