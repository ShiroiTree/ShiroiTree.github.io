# T003 数据类型大小及范围

每一个 C/C++ 整形，都有一个内存大小和数值范围。例如：int，内存大小为 4，数值范围为（-2147483648,2147483647）。

变量占用的内存大小可以用运算符 sizeof () 得到，变量的数值范围可由乘方运算得到。为简便运算，此处代码使用预先计算的方式。

# T004 平均值

编写一个程序，输出整数 A、B 的平均值。

由输入输出条件及样例可知，此处结果不包含浮点部分，故直接计算。

# T005 进制转换

输入一个非负整数，输出它的十六进制，八进制。

该题要求将一个十进制整数转换为十六进制和八进制，原则上，我们可以取模、相除后再倒序输出结果，对于十六进制，可以使用一个表，保存字母部分。但是不论如何，这种办法过于复杂。
在 C 语言中，系统已经预留了将变量按照不同进制输出的占位符，八进制 % o ，十进制 % d ，十六进制 % x (x 小写则输出小写字母形式的十六进制数，X 大写则大写形式)。
若在一段给定的数据 1234567，无标识符则默认为十进制，0 表示八进制，0x 表示十六进制，如 01234567，0x1234567。

# T006 浮点数输出

输出一个浮点数的 3 个格式结果

对于浮点数，使用标准输入输出占位符可以输出指定位数的小数。

如 1234567.1234567

我用 %.4lf 输出可得到，1234567.1235，程序将自动保留对应的小数，并四舍五入。
